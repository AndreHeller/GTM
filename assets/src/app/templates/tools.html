Doporučené nástroje

Seznam nástrojů pro efektivnější práci s Google Tag Managerem.

Google Chrome Browser

Webový prohlížeč. Jeho důvodem je kompatibilita s dalšími pluginy.
http://bit.ly/GoogleChromeBrowser

Ghostery

Plugin pro základní monitorování spuštěných kódů.
http://bit.ly/ghosteryplugin

WASP.inspector - Web Analytics Solution Profiler

Plugin pro zobrazování a parsování měřících knihoven a kódů a jejich závislostí na základě HTTP hlaviček.
http://bit.ly/pluginwasp

dataSlayer

Plugin pro zacyhtávání HTTP hlaviček.
http://bit.ly/gtmdataslayer

Chrome Developer Tools

Nástroj pro vývojáře zabudované v prohlížeči.
Stiskněte klávesu F12 nebo kombinaci kláves Ctrl + Shift + J.

  • Tag Manager Validator - vypisuje dataLayer

Usnadnují práci s GTM rozhraním

  • GTM Copy Paste - umožnuje kopírování jednotlivých entit mezi kontejnery
  • SANDOGTM - Upravuje rozhraní

  • Code Editor for GTM - alternativní highligter pro GTM custom kódy, funguje střídavě
  • Tag Manager Injector - Umí do stránky vložit cizí GTM
  • GTM Sonar - pomáhá odhalit problém auto eventů a jQuery
  • Da Vinci Tools - spousta funkcí pro GTM a GA
